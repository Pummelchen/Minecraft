# Pummelchen Tropical Worldgen

This datapack overrides Terralith's Lithostitched overworld biome parameter list.
It redirects selected warm land climate cells into bamboo jungle, jungle,
Terralith tropical jungle/rainforest variants, and Terralith sakura valley.
Oceans, rivers, beaches, swamps, caves, and existing jungle/cherry cells are not rewritten.

- Changed entries: 313 of 1713
- Tropical rough climate-volume factor: 5.42x
- Cherry/sakura rough climate-volume factor: 2.81x
