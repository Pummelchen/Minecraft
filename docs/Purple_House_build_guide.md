# Purple House Build Guide — Minecraft Java 26.1.2

## Exterior first
Use the data pack function:
`/function purple_house:exterior`

This builds:
- raised deepslate terrace
- lower dark-oak basement level
- main white-quartz wall body
- upper left and right wings
- steep purpur/purple roof gables
- front balcony, lanterns, stairs, warm glass windows
- chimney tower and amethyst ridge accents
- purple crystal garden and planters

## Interior second
Use:
`/function purple_house:interior`

This adds:
- basement storage/workshop/crystal cellar
- main foyer, living room, fireplace, kitchen, dining area
- staircase to second floor
- bedroom with balcony access
- library/enchanting room
- attic reading/observatory lofts under the gables

## Full build
Use:
`/function purple_house:complete`

## Safety
The exterior function clears a large volume before building. Back up your world first.
